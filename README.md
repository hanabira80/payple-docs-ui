# 페이플 Docs Renewal 

## 환경설정
- 사용 Framework
> vue cli   

- 사용 Plugin
> vuetify

- 프로젝트 Setup
```
npm install
```

- 프로젝트 개발서버
```
npm run serve
```

- 프로젝트 배포 
```
npm run build
```

## 업데이트
- 2020.06.02


## 기타
> vue를 이용한 자사 서비스 DOCS페이지를 제작하는 프로젝트 입니다.   
퍼블리셔(한국한정)로 분들에게 프론트 프레임워크에 진입하는 가이드가 됐으면 합니다.   
html, css, jquery(데이터 핸들링 제외)를 기본 스킬로 하시는 분들을 대상으로 합니다.
